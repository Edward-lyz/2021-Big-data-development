### 第三周任务
基于kafka的订阅分发机制进行和S3相配合的流式计算，具体代码和实现在文件夹“课程三：流式计算”下
### 第四周任务
基于scikit-learn的基本回归算法对手写数字数据集进行训练和预测，具体代码和实现在文件夹“课程四：让人工变智能”下
